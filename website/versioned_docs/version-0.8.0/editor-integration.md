# Editor Integration

agnix ships editor integrations around the `agnix-lsp` server.

## LSP server

```bash
cargo install agnix-lsp
```

Main capabilities:

- Diagnostics on open/save/change
- Code actions for fixable findings
- Hover details for rule explanations
