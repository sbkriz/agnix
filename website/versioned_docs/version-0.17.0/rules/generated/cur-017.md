## Examples

The following examples demonstrate what triggers this rule and how to fix it.

### Invalid

```json
{
  "hooks": {
    "save-file": [{ "command": 42, "events": "file_saved" }]
  }
}
```

### Valid

```json
{
  "hooks": {
    "save-file": [{ "command": "npm run lint", "events": ["file_saved"] }]
  }
}
```
